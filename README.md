# Loan Eligibility Predictor

## Overview

This project predicts whether a loan application will be approved using Machine Learning.

The prediction is based on applicant information such as:

- Gender
- Marital Status
- Income
- Education
- Credit History
- Loan Amount

## Features

- Data Cleaning
- Missing Value Handling
- Label Encoding
- Data Visualization
- Model Training
- Prediction
- Model Saving

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm

Random Forest Classifier

## Accuracy

Approximately 80–90% (depends on dataset)

## Run

python main.py
